1. git init -> создание файла настройки .git 
2. git add <filename> -> добавление файла на загрузку
2.1. git add . -> добавить все файлы в папке на загрузку 
3. git commit -m '<message>' -> команда потверждает измение файлов и окончательно готовит их к загрузке
4. git remote add <link name> <ssh link> -> Пишется один раз после git init , добавляет ссылку на репозиторий в конфигурацию
5. git push origin <branch name> -> загрузить измения, совершенные в git commit на сайт

Для загрузки новых файлов повторить шаги (2,3,5)
